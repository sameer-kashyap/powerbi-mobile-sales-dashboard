# 📊 Mobile Sales Dashboard – Power BI

This repository contains an **end-to-end Power BI project** built using a Mobile Sales dataset.  
The project covers everything from **data cleaning to advanced DAX and interactive dashboarding**.

This dashboard was created by following and learning from an excellent tutorial by **Satish Dhawale**, which helped me revise and strengthen my Power BI fundamentals.

---

## 🚀 Project Overview

The goal of this project was to build a **complete BI solution** that answers key business questions related to mobile sales performance across:

- Time (MTD, Monthly, Yearly)
- Geography (City-wise sales)
- Brands & Mobile Models
- Payment Methods
- Customer Ratings

---

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **Power Query (ETL)**
- **DAX**
- **Data Modeling**
- **Interactive Visuals & Slicers**

---

## 🔄 ETL Process (Power Query)

- Imported raw mobile sales data
- Cleaned and transformed data
- Handled data types, null values, and formatting
- Created a **custom calendar table**

---

## 🧠 Data Modeling

- Built relationships between fact and dimension tables
- Implemented a star schema
- Created a dedicated Date table for time intelligence

---

## 📐 DAX Measures Used

Some key DAX concepts implemented:

- `Total Sales`
- `Total Quantity`
- `Average Price`
- `TOTALMTD`
- `SAMEPERIODLASTYEAR`
- Year-over-Year comparisons
- Month-to-Date (MTD) analysis

---

## 📊 Dashboards Included

### 1️⃣ Main Sales Dashboard
- Total Sales, Quantity, Transactions
- Sales by City (Map)
- Sales by Mobile Model & Brand
- Payment Method analysis
- Day-wise sales trend

### 2️⃣ MTD Report
- Month-to-Date sales progression
- Daily sales accumulation
- Dynamic filtering by month and year

### 3️⃣ Same Period Last Year Analysis
- Year-wise comparison
- Quarter-wise comparison
- YoY performance insights
